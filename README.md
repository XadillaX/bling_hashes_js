String hash algorithms for node.js.

## Installation

```bash
$ npm install --save bling-hashes-js
```

## Introduce & Usage

Refer to https://github.com/XadillaX/bling_hashes#introduce and repace `bling-hashes` by `bling-hashes-js`.
